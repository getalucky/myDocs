    html

    在html文件中，结构实际上是有两种结构关系

    1. 并列关系----兄弟关系
2. 嵌套（包含）关系----父子关系

    用emmet语法快捷生成html基本骨架

    * 后缀名必须是html

* `!`(英文符号) + enter

* `<!DOCTYPE html>` 文档声明类型标签，该文档是按照html5的版本去解析

* ' <html lang="en"> ' language语言 en - English , zh-CN - 中文

* ' <meta charset="UTF-8"> ' 字符编码格式

* ' <meta name="viewport" content="width=device-width,initial-scale=1.0"> ' 浏览器视口，'width = device-width'(设备宽度)，'initial-scale=1.0'(初始化缩放)

* ' <meta http-equiv="X-UA-Compatible" content="ie=edge"> ' 在windown系统中使用ie最高版本的浏览器edge打开

* 虽然一些内容写在body外也能显示，但还是推荐大家写在body标签中方便管理

    翻译会导致出现非自己写的元素

    emmet 便捷语法 >> emmet中*是批量生成对应结构 ：h16 >> $代表自增的正整数 ： h$6 快速生成递增的结构 >> 在标签中插入文字内容 {}插入内容 ： h1*6{我是h$标签} >> + 生成兄弟标签

    标签

    标题标签(h1~h6)的特点：

    1. 随着字体的增大，字体大小会变得越来越小，甚至会小到比默认字体大小还小。

2. 所有标题标签里面的内容都会加粗显示。

3. 标题标签独占一行。

4. 并且行与行之间的间距会变得很大（默认的外边距引起的）

5. 在实际开发过程中，Web规范规定一个网页只允许出现一个' <h1> '标签。就是为了强调当前网页的主题内容不被其他内容混淆。提高搜索引擎搜到我们网页的概率，提高网站的排名。

    段落标签(p)

    1. 段落标签实际上就是处理文章中段落的标签

2. 还可以换行显示段落

3. 不同段落之间有较大的间距

    一大段没有空格的无意义的数字或者是字母集合会被当成一个整体，不进行换行 全角空格是不会受浏览器合并多个空格和换行影响的。全角空格是一个中文字符的大小。

    换行标签(br单标签)

    1. html中遇到' br '标签会强制换行

2. 与段落标签不同的是，' br '的换行不会有较大的间隔，就像普通的文本换行一样去显示

    水平线标签(hr单标签)

    1. 用来区分不同结构的水平线分割

    文本格式化标签

    * ' <strong></strong> ' (语义更强烈) / ' <b></b> ' 加粗

* ' <em></em> ' (语义更强烈) / ' <i></i> ' 倾斜

* ' <del></del> ' (语义更强烈) / ' <s></s> ' 删除线

* ' <ins></ins> ' (语义更强烈) / ' <u></u> ' 下划线

    无语义化但很重要的标签

    div

    1. 块级元素，独占一整行

2. 上下正常间距。

    span

    1. 行内元素，可以和其他行内元素在一行显示

2. 宽度由内容撑开。

    多媒体标签

    img标签

    * 'src'属性 路径

> 路径
>> 绝对路径：
>>> 硬盘绝对路径
>>> 网络绝对路径

>> 相对路径
>>> 目标和自身同一文件夹，直接写文件名 或 ' ./(文件名) '
>>> 目标文件在自身文件夹的下一级  ' /(文件名) '
>>> 目标文件在自身文件夹的上一级  ' ../(文件名) '
>>> 到了盘符目录继续' ../ '没有效果
>>> ' / '作为路径开始的概念只有服务器中或mac系统(也是配合同级一起使用)

* ' width '属性 宽度，像素单位（不用带单位）

* ' height '属性 高度，像素单位（不用带单位）

* ' alt '属性 替换文本，通常是图片无法正常加载时显示出来

* ' title '属性 鼠标悬停时候会显示的文字（任何标签都有' title '属性）

## 解决图片底部默认空白缝隙问题

1. 给图片添加 vertical-align:middle | top| bottom 等。 （提倡使用的）

2. 把图片转换为块级元素 display: block

3. 给父元素设置`font-size`

4. 给父元素设置`line-height`

    只写一个值width/height，width/height会等比例缩放.

    超链接标签(a)

    1. 跳转到其他html页面的一个标签

2. ' href '属性 跳转的路径

    1. 跳转到网络的绝对路径

    2. 跳转到本地绝对路径

    3. 跳转到相对路径

    4. 空链接，' # '代替链接地址

    5. 下载链接，地址链接是文件' .exe ' 或者是 ' .zip '等压缩包形式

    6. 描点链接：点击可以快速定位到页面中的某个位置
        * 在连接文本' href '属性中，设置属性值为 #名字 的形式，如' <a href="#two">第二集</a> '
        * 找到目标位置的标签，里面添加一个id属性 = 刚才的名字，如' <h3 id="two">第二集介绍</h3> '

3. ' target '属性

* ' target = "_self" ' 自身标签页面跳转 默认值

* ' target = "_blank" ' 新建一个标签页打开要跳转的页面

    在里面加上h5新增语义化标签 

    * <header>：头部标签
* <nav>：导航标签
* <article>：内容标签
* <section>：定义文档某个区域
* <aside>：侧边栏标签
* <footer>：尾部标签

    注意点

    1. 这种语义化标准主要是针对搜索引擎的  
2. 这些新标签页面中可以使用多次
3. 在 IE9 中，需要把这些元素转换为块级元素
