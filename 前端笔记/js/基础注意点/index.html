    匿名函数

    没有名字的函数被称为匿名函数，匿名函数是无法独立存在的。函数的目的就是为了能够让代码被重复使用。函数要调用才能使用。如果没有名字就无法调用了。所以不允许匿名函数独立存在。

    自调用函数

    自动调用的函数

    (funciton(){
    // 你希望立刻被调用的代码 
 })();

    函数中的小细节

    1.函数的内部可以调用另一个函数

    2.函数被调用的时候，会回到他声明的位置继续执行

    3.当函数被调用完毕，会回到它被调用的位置继续向下执行

    如果要在浏览器里面跳过函数的调用，我们可以使用F10快捷键

    4.函数的参数的个数 ： 形参和实参的个数是可以不一致的。如果有少传递的参数，默认就是undefined

    此时我们可以使用 短路运算给参数写默认值

    function fn(a,b,c){
  a = a || 0;
  b = b || 0;
  c = c || 0;
  console.log(a + b + c);
}
fn(10,20);
fn(10,20,30);

    5.函数可以作为参数传递 ， 哪些东西可以作为参数 —— 只要是数据类型就可以

    6.函数也是一种数据类型 —— typeof 函数 === 'function'

    7.通过逻辑与的短路运算可以给函数简化判断参数是否是传进来的函数

    箭头函数

    总结:

    1. 箭头函数没有自己的this, 由函数定义时的上下文决定(就是定义箭头函数时所在的作用域)。
    2. function 函数内部有自己的this,还受到函数调用者的影响
    3. 箭头函数没有arguments对象
    4. 箭头函数不可以作为构造函数8主要还是因为没有this的影响
    5. 其他：箭头函数直接返回一个对象，必须在对象外面加上括号，否则会报错。
    const fn = () => ({name: 小明});
    1. 重点注意：箭头函数没有自己的this,所以在箭头函数中的this 不指向调用者。

    预解析

    代码在执行的时候，不是简单的解析一行执行一行，而是有一个提前解析的过程 —— 预解析

    预解析做了：

    1.把函数和变量的声明，提升到当前作用域的最顶端

    2.函数的调用以及变量的赋值，还在原来的位置。

    预解析之后，会变成下面的代码

    function a(){
  console.log(1);
}
var b;
a();
a();

b();
b = function(){
  console.log(2);
}
b();

    预解析之后，会变成下面的代码

    function a(){
  console.log(1);
}
var b;
a();
a();

b();
b = function(){
  console.log(2);
}
b();

    arguments伪数组

    如果希望实现一些参数个数是不一定的情况，就可以使用arguments伪数组来实现

    是一个js的函数里面自带的东西，不需要声明，直接在函数内部使用，它代表了所有的函数被调用的时候的实参。绝对不要在函数的外面使用。

    function add(a,b){
  // 就可以遍历这个伪数组，把里面的所有的实参都拿出来，再累加
  var sum = 0;
  for(var i = 0; i < arguments.length; i++){
    sum += arguments[i];
  }
  // console.log(sum);
  return sum;
}

    就可以求任意个数的数字的和

function Student(name,age,gender){
