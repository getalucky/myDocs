    float浮动

    特性

    1. 脱离标准流

      • 脱离标准流移动到指定位置

      • 浮动的盒子不再保留原先的位置

    2. 一行内显示并且元素顶部对齐

      • 浮动元素互相靠在一起不会有间隙

      • 如果父级宽度装不下浮动的盒子，多出的盒子会另起一行

    3. 具有行内块元素的特性

      • 任何元素都能浮动，添加浮动后具有行内块相似的特性

      • 浮动元素本质上转换成了块级元素，表现出来的是行内块元素的特性

      • 如果行内元素有了浮动，则不与要转换成块级/行内块就可以直接给高宽

      • 如果块级盒子没有设置宽度，默认和父级一样。但是添加浮动后，它大小根据内容决定

    4. 浮动不会压住下面标准流的文字

    清除浮动

    清除浮动的本质是清除浮动元素造成的影响 如果父盒子本身有高度，则不需要清除浮动 清除浮动之后，父级就会根据浮动的子盒子自动检测高度。父级有了高度，就不会影响下面的标准流了

    position定位

    1. static 静态定位，无定位的意思

    2. absolute 绝对定位

      • 元素移动时，相对于它的祖先元素来移动偏移值
      • 如果没有祖先元素或者祖先元素没有定位，以浏览器为准来定位
      • 如果祖先元素有定位(相对，绝对，固定定位)，则以最近一段有定位祖先元素为参考点移动位置
      • 不占有原来的位置(脱离标准流)

    3. relative 相对定位

      • 元素移动位置的时候，是以它原来的位置来移动偏移值
      • 保留原来的位置，后面的盒子不会占有其位置(不脱离标准流)
      • 可以限制绝对定位

    4. fixed 固定定位

      • 固定于浏览器可视区的位置。主要用于浏览器滚动界面时位置不变的元素
      • 以浏览器的可视窗口为参照移动元素
        • 跟父元素没有关系
        • 不随滚动条滚动
      • 不占有原先的位置(脱离标准流)

    5. sticky 粘性定位 (不常用)

      • 以浏览器的可视窗口为参照点移动元素
      • 占有原先位置
      • 必须添加 top left right bottom 其中一个才有效

    6. z-index : 整数 层级

      • -∞ ~ +∞。数值越大，盒子越靠上
      • 属性值相同，按书写顺序排列，后来居上
      • 数字后面不加单位
      • 只有定位的盒子才有这个属性

    7. 行内元素加固定定位、绝对定位可以设置宽高

    8. 块内元素不给宽高，默认大小是内容大小

    定位 = 定位模式+边偏移

    定位模式 ：上面1~5 边偏移 ： top , bottom , left , right （不是相同属性不会层叠）

    脱离标准流的特性

    • 1. 脱离标准流的元素，会隐式转换成 display:block;
    • 2. 脱离标准流的块级元素,默认宽度不再是一整行，而是内容宽度
    • 3. 不会触发margin塌陷
    • 4. 脱离标准流的元素，不能通过margin:auto;进行居中
