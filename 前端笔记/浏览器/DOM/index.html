    DOM

    DOM（Document Object Model）：文档对象模型

    事件

    事件三要素

    • 事件源: 被触发事件的元素，如 当按钮被操作
    • 事件类型: 事件的触发方式，如 鼠标点击 或 键盘输入
    • 事件处理函数: 事件触发后要执行的代码，如 函数内让 body 变背景色

    事件的基本使用

    // 1. 查找事件源
var box = document.getElementById('box');
// 2. 给事件源添加点击事件
box.onclick = function() {
    // 3. 事件处理函数内部执行的代码
    console.log('代码会在box被点击后执行');  
};

    获取,修改标准属性

    修改：
元素.标签属性名  =  值;

获取：
console.log(元素.标签属性名)   //输出元素属性值

    href、title、id、src、className

    特别注意，类名属性的获取和修改需要写成 classNa

    只能获取设置行内样式

    表单属性操作

    表单属性

    表单常见属性

    • value 用于大部分表单元素的内容获取
    • type 可以获取input标签的类型

    3 个特殊表单属性，获取的是布尔类型数据。

    • disabled 禁用属性
    • checked 复选框选中属性
    • selected 下拉菜单选中属性
    // 获取到布尔类型数据,设置的时候也是布尔类型。
元素.disabled = true;
元素.disabled = false; //可以直接设置创建属性
    1. 普通属性获取到的是字符串类型，设置的时候也是字符串类型数据。如 id,title,src,type,value。
    2. 类名获取和设置需要写成 className。
    3. 表单 disabled ，checked， selected 这3个特殊属性，获取类型是布尔类型，设置的时候也是布尔类型。

    获取、创建标签内容

    innerHTML 和 innerText

    元素.innerHTML //获取标签里所有内容。 可以创建标签，带标签格式
元素.innerText // 获取标签里文本内容。 只可以创建文本，如果带标签不被解析
    • innerHTML 直接赋值会覆盖内容

    模板字符串(ES6)

    我们需要用反引号

    `.....${数据}........`

    将内容引起来，对于用数据填充的地方用${数据}

    自定义属性

    <input data-id="11" data-text="你好" type="button" value="按钮">

data-属性名        开头的我们叫自定义属性

    获取和设置自定义属性语法

    获取

    元素.dataset.属性名

    注意事项：获取时候返回的数据类型都是字符串形式。

    设置

    元素.dataset.属性名 = 值;

    注意事项：1. 不管赋值的时候是什么数据类型，JS 都会自动转换成字符串类型。

    ​ 2. 如果这个属性不存在，会在元素上添加这样一个属性

    删除

    delete 元素.dataset.属性名

    注意事项：delete 是对象删除属性的通用语法，其他对象也能通过 delete 关键词删除掉某个属性。

    元素的操作标签属性的方法(常用)

    功能 方法名 参数个数 是否有返回值
    获取属性 getAttribute() 一个参数：属性名作为参数 有返回值
    设置属性 setAttribute() 两个参数：属性名 和 属性值 undefined
    移除属性 removeAttribute() 一个参数：属性名作为参数 undefined
    是否有属性 hasAttribute() 一个参数：属性名作为参数 有返回值，布尔类型

    注意事项

    • 以上方法也可以获取，修改，移除操作标签属性。
    • 最大的差别就是方法需要注意函数三要素：方法名，方法参数，方法是否有返回值。
    • 以上方法可以操作标准属性，也可以用于操作自定义属性，如：元素.getAttribute("date-id")。
    • setAttribute()可以添加新的属性。
    • 参数中的属性名应写成字符串类型。

    管理类名的对象classList

    功能 方法名 参数个数 是否有返回值 重点
    添加 element.classList.add() 一个或多个类名，逗号隔开 **
    移除 element.classList.remove() 一个或多个类名 **
    切换 element.classList.toggle() 一个或多个类名 *
    替换 element.classList.replace() 旧类名，新类名
    是否包含 element.classList.contains() 一个类名 布尔类型
    • element.classList.add()返回一个伪数组，参数一定是字符串。如果元素已经存在同样的类，直接忽略。
    • element.classList.remove()如果删除一个元素中没有的类，直接忽略。
    • element.classList.replace()替换成功返回true，替换失败返回false。
    • element.classList.toggle()如果只有一个参数，如果类值存在，则删除它并返回false。如果不存在，添加类值，返回true。

    动态创建元素createElement

    createElement() 创建元素

    document.createElement('标签名')

    • 在内存创建新的标签，但是不会显示在页面中

    • 如果要在页面中显示，还需配合 JS 负责添加到页面的方法才可以

    节点操作

    添加子节点

    element.appendChild(子节点)；
    • element 就是作为父元素对象。要添加的元素其实是作为子元素的方式添加。
    • 追加的元素都在最后面。

    创建标签三种方式

    document.createElement('标签名')

    常用于创建一个新的标签，并后添加到页面的情况，如：消息发布，聊天

    element.innerHTML = '标签字符串';

    适用于快速创建一个默认列表或内容，后追加就不太合适，因为 innerHTML 会覆盖原内容。

    document.write('标签字符串');

    了解即可，因为只能在 body 后添加内容而已，局限性太大。

    移除子节点

    父元素.removeChild(子节点);

    • removeChild 方法需要由父元素调用，才可以移除 子元素。

    • 节点被移除后，在页面中完全消失不见了.

    • 返回值是删除的元素。

    替换子节点

    父元素.replaceChild(新节点, 旧节点);
    • 旧节点在页面中消失了，但是在内存还有的，返回值为替换的节点。

    插入节点

    父元素.insertBefore(子节点, 位置);

    • 第二个参数为插入位置，当然也是一个 元素对象。

    • 如果位置参数为null，新节点插入在最后

    克隆节点

    被克隆的节点.cloneNode(布尔值)
    • 返回被克隆出来的节点
    • 不传参数仅仅只克隆当前一层标签
    • 参数为true，包括克隆下面所有的节点和内容。参数为false，只克隆当前标签

    选取元素关系

    • 元素.children 子元素集合，伪数组

    • 元素.parent 父元素/父节点，单个值

    • sibling 兄弟关系

    • 元素.previousElementSibling 获取上一个兄弟元素。
    • 元素.nextElementSibling 获取下一个兄弟元素

    事件类型

    鼠标事件

    使用时要：on事件类型

    click       //点击

    //如果有遮罩层(分区内部有元素遮挡了指针)，会多次触发事件
mouseover   //移入
mouseout    //移出

    //因为上面两个有问题所以出现了下面两种。推荐使用 
mouseenter  //移入
mouseleave  //移出

    mousemove   //鼠标移动

    鼠标事件对象属性

    clientX     //鼠标在可视区的 X 轴坐标
clientY     //鼠标在可视区的 Y 轴坐标

    pageX       //整个页面的X坐标
pageY       //整个页面的Y坐标
    • 如果元素是全屏幕移动，使用 clientX/clientY 更合适，配合 position: fixed; 更合适，如鼠标跟随效果。
    • 如果元素是在页面某个区域内移动，使用 pageX/pageY 更合适，配合 position: absolute; 更合适，如放大镜效果。

    键盘事件

    keydown     //键盘按下
keyup       //键盘弹起
keypress    //键盘按下并弹起

    键盘事件对象属性

    keyCode     //获取键盘按键编码

    keyCode : 13（回车）

    ctrl+enter时，keyCode : 10（回车），但有时候会是13

    表单事件

    focus       //获取焦点
blur        //失去焦点
submit      //提交表单
change      //改变域

    事件对象的通用属性

    属性 描述 具体
    type 事件类型 当前执行的事件的事件类型
    timeStamp 时间标记 当前执行这个事件的开始时间点，单位是 毫秒
    target 事件目标元素 触发事件时候真正被(点击)那个元素（ 重点

    事件对象方法

    方法名 描述 备注
    preventDefault() 阻止默认行为
    stopPropagation() 停止事件传播

    默认行为

    • <input type="submit"> 按钮点击的时候，会提交 form 表单。
    • <a href=""></a> a 链接点击的时候，会自动跳转到 href 的地址。
    • 在浏览器界面中，点击鼠标右键会弹出菜单。
    • 在浏览器界面中，Ctrl + C 复制内容。

    应用场景：

    • 用户名，密码格式不正确，阻止表单提交。
    • 百度网盘用户没有登录，阻止 a 链接下载文件。
    • 部分网站，禁用用户鼠标右键菜单，禁用用户复制内容。

    事件传播

    事件冒泡传播

    • 每个元素触发时候，事件会自动往父级元素冒泡传播。
    • 如：子元素添加了点击事件，父元素也添加了点击事件，子元素点击触发后，会继续触发父元素上的点击事件。
