    jQuery基础

    概念

    jQuery 是一个 JavaScript 库，其实就是通过 原生JS 封装了的很多的 方法 和 属性。

    主要封装操作 DOM 的一系列方法，也就是 document 对象下的属性和方法，元素的属性和方法。

    jQuery 选择器

    jQuery 选择器非常强大，同一个元素可以有很多种查找方式。

    全面支持 CSS 3的选择器，而且浏览器兼容非常好。

    查找第一个 li 元素:

    $("li:eq(0)")
$("li").eq(0)
$("li:first")
$("li:first-child")
...

    将JQ对象转换成DOM对象:

    JQ对象[索引]

    将DOM对象转JQ对象

    $(DOM对象)        //将原生DOM对象通过JQ进行一次包装，包装成JQ对象

    css()方法

    jq对象.css('width')   //只有一个参数，获取元素的属性值

jq对象.css('width',200/"200px")   //两个参数，可以给元素的属性设值(设置行内样式)
//第二个参数是数字或者布尔值的时候可以不用引号，如果是字符串一定需要引号包起来

jq对象.css({      //如果要设置多个值，可以传入一个对象
    width: 20,
    height: "20px"
})

    入口函数

    入口函数写法

    写法1

    $(function(){

})

    写法2

    $(document).ready(function(){

})

    写法3 了解

    $().ready(function(){

})

    等页面上的东西加载完，然后执行入口函数中的方法。等页面元素加载完成即可(标签，页面的DOM树结构完成)

    事件注册

    JQ对象.on("事件类型" , 事件处理程序(){})

    • 不会造成事件覆盖

    • 方法内部已经遍历

    jQuery 很多方法内部都会有遍历，直接调用即可。

    $()      循环选中所有符合规则的选择器

.on()    把所有的JQ对象遍历添加事件

.css()   jQuery内部也有遍历
    • 在遍历的时候获取索引
    • 单个元素也会有索引
    $('ul li').on("cilck",function(){
    console.log($(this).index());   //$(this).index() 可以获得索引
})

    事件解绑

    事件.off('事件类型');

事件.off('事件类型',fn);
    • 一般解绑不用两个参数的用法

    事件委托

    jQ对象.on("事件类型","需要触发的元素的选择器/子元素",functon(){ })

    通过关系查找元素

    关系
    父子 parent()只能找到元素的父级元素
    parents()找到所有的父辈元素
    兄弟 siblings()找到这个元素的所有的兄弟关系元素
    孩子 children()找到这个元素的所有孩子元素

    筛选

    $('标签选择器').eq(索引)   //根据索引筛选获得对应元素
//索引 -1 是选择最后一个(倒数从-1开始)， -2 选择倒数第二个

    属性操作

    方法 备注
    prop('属性' , ‘属性值’) 一个参数获取第一个满足匹配的标准属性值。两个参数修改属性值
    removeProp() 删除属性
    attr() 可以操作标准属性, 自定义属性。
    removeAttr() 删除属性

    • 设置多个属性可以传一个对象参数

    • attr() 设置多个属性 - 传入 对象格式 属性和值

    js .attr({ 属性1: "值1", 属性2: "值2" });

    便捷写法

    jQuery 方法 描述 备注
    .html() 获取和设置 盒子的 HTML 内容 可通过 innerHTML 封装
    .text() 获取和设置 盒子的 文本内容 可通过innerText封装
    .val() 获取和设置 表单 值 可通过value封装

    三个方法不传入参数都代表获取，传入参数都代表设置。

    类名操作

    类名 功能 备注
    $('标签选择器').addClass() 添加类名 可以添加多个类，但要空格隔开
    $('标签选择器').removeClass() 删除类名
    $('标签选择器').toggleClass() 切换类名
    $('标签选择器').hasClass() 检测类名 返回Boolean值

    文档处理

    创建新节点

    创建一个 新的 li

    $('<li>')

    创建一个 新的 li 并写入内容

    $('<li>内容</li>')

    子级节点添加

    方法名 参数 备注
    .append() 可传入节点或直接传入 html 字符串 .appendChild() 封装而成的
    在原有的元素的孩子元素的后面追加一个
    .prepend() 同上 .insertBefore() 封装而成的
    在原有的元素的孩子元素前面添加新内容
    .appendTo() $(A).append(B) 的颠倒写法 str.appendTo($('ul')); //将str添加到ul的后面
    .prependTo() $(A).prepend(B) 的颠倒写法 str.prependTo($('ul')) // 将str添加到ul的前面

    同级节点添加

    方法名 参数 备注
    .after() 可传入节点或直接传入 html 字符串 同级后添加
    .before() 同上 同级前添加
    .insertAfter() A.insertAfter($('ul')) 将哪一个元素添加到某一个元素的后面（兄弟关系）
    .insertBefore() A.insertBefore($('ul')) 将哪一个元素添加到某一个元素的前面（兄弟关系）

    删除节点

    方法 参数 备注
    .remove() 可传入选择器 将元素在页面中彻底删除
    .detach() 可传入选择器 删除元素，但是元素身上的属性，方法会保留在内存中
    .empty() 删除匹配元素的文本内容

    替换节点

    方法名 参数 备注
    .replaceWith() 新节点 基于 replaceChild() 封装
    .replaceAll() $(A).replaceWith(B) 的颠倒写法 了解

    克隆节点

    方法名 参数 备注
    .clone() 传入 布尔值 true 或 false 基于 cloneNode() 封装
    将元素克隆了一份，放在内存中

    注意实现：

    .clone() 不管是否传参数都把内容克隆了，如果传入 参数 true，会把元素的事件也克隆了一份，内部帮我们绑定了对应的事件。

    循环遍历.each()

    jq对象.each(function(索引 , 遍历的每个对象){}) //遍历得到的每个对象都是原生dom对象
    • 跟原生的 forEach(function(遍历的每个对象 , 索引){ }) 括号里的函数里的参数正好相反

    动画

    jq的动画在执行的时候，传入的参数有四个

    1. 第一参数便是动画效果的属性，必须是一个对象，对象中的属性以键值对的方式出现。哪怕是一个属性也需要这么做(必写)

    2. 只有值是数字+px作为单位的属性才能实现动画效果

    3. 第二参数是预定速度的字符串("slow","normal", or "fast")，或者表示动画时间的毫秒数值(可不写)

    4. 第三个参数是要使用的擦除效果的名称(需要插件支持).默认jQuery提供"linear" 和 "swing".(可不写)
    5. 第四个参数是在动画完成时执行的函数，每个元素执行一次

    滚动偏移

    方法名 描述 参数 返回值
    .scrollTop() 顶部滚动出去了多少像素 可获取和设置 数值型
    .scrollLeft() 左边滚动出去了多少像素 同上 同上

    页面滚动跳转

    滚动页面写法1：直接跳转。

    $(window).scrollTop(数值型);

    滚动页面写法2：带动画效果跳转

    $('html,body').stop().animate({ scrollTop : 数值型 });

    元素偏移值

    方法名 描述 参数 返回值
    .offset() 获取或设置元素在页面的偏移值 传入对象可以设置偏移 返回偏移对象{ left:xx, top:xx }
    .position() 获取或设置元素在定位父元素的偏移值 没有参数 返回偏移对象{ left:xx, top:xx }

    注意事项：

    • .offset() 可以传入对象设置 { left:xx, top:xx } 偏移值，不传入参数代表获取偏移值，.offset() 是基于整个页面的偏移。
    • .position() 只能获取偏移值，.position() 受定位父级元素影响。
